#### ✅ 현재 우리 페이지에 있는 기능
- 방문예약 신청 링크
- 예약현황 링크
- 공지사항, FAQ 링크

#### ❌ 추가 필요한 기능

**1. 회사 선택 기능**
- 한미제약: 회사 선택 드롭다운 (한미사이언스, 한미약품, 온라인팜, 뮤지엄한미, 코리/오브맘코리아)
- **개선점**: 광동제약도 여러 회사/법인을 운영할 수 있으므로 회사 선택 기능 추가
- **구현**: `companies` 테이블 생성 및 선택 기능 추가

**2. 지역 선택 기능**
- 한미제약: 지역 선택 드롭다운
- **개선점**: 광동제약도 여러 지역(본사, 공장, 연구소 등)이 있을 수 있으므로 지역 선택 기능 추가
- **구현**: `locations` 테이블 생성 및 선택 기능 추가

**3. 방문절차안내 시각화**
- 한미제약: STEP 01-05를 시각적으로 표시
- **개선점**: 현재 우리 페이지에는 없음. 홈페이지에 방문절차 안내 추가 필요
- **구현**: 홈페이지에 5단계 프로세스 시각화 추가

**4. Breadcrumb (경로 표시)**
- 한미제약: HOME > 예약현황 형태의 경로 표시
- **개선점**: 현재 우리 페이지에는 없음. 사용자 경험 개선을 위해 추가
- **구현**: Breadcrumb 컴포넌트 추가

### 2. 예약현황 페이지

#### ✅ 현재 우리 페이지에 있는 기능
- 방문자명, 전화번호, 예약번호 검색
- 테이블 형태로 결과 표시
- 방문 취소 기능

#### ❌ 개선 필요한 사항

**1. 검색 필드 레이아웃**
- 한미제약: 더 깔끔한 레이아웃, "방문자 정보" 섹션으로 그룹화
- **개선점**: 현재 우리 페이지도 비슷하지만 레이아웃 개선 가능

**2. "데이터가 존재하지 않습니다" 메시지**
- 한미제약: 테이블이 비어있을 때 명확한 메시지 표시
- **개선점**: 현재 우리 페이지에도 있지만 스타일 개선 가능

**3. 전체 건수 표시**
- 한미제약: "전체 (0)" 형태로 표시
- **개선점**: 현재 우리 페이지에도 있음 (Badge 형태)
