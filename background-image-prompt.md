# 배경 이미지 생성 프롬프트

## 기본 정보
- **용도**: 방문예약 신청 폼 배경 이미지
- **브랜드 컬러**: KD Red (#FF1A00, rgb(255, 26, 0))
- **이미지 크기**: 1920px × 1080px (16:9 비율) 또는 1600px × 900px
- **파일 형식**: WebP 또는 PNG (투명도 불필요 시 JPG)
- **파일 크기**: 200KB 이하 권장

## 프롬프트 (영문)

```
Create a modern, clean, abstract background image for a pharmaceutical company's visit reservation website. 

**Style & Aesthetic:**
- Modern, minimalist, professional design
- Abstract 3D geometric shapes and forms
- Low-poly or faceted polygonal surfaces
- Subtle wireframe network elements
- Clean white or very light background
- Digital, architectural, scientific aesthetic

**Color Palette:**
- Primary color: Bright red (#FF1A00, rgb(255, 26, 0)) - "KD Red"
- Use varying shades of red: lighter orange-red highlights, deeper dark red shadows
- Subtle gradients across the red surfaces
- Light grey wireframe lines and nodes (very subtle)
- Pure white or off-white background (#FFFFFF or #FAFAFA)

**Main Elements:**
- Central 3D geometric shape: A continuous, ribbon-like form that twists and loops
- Shape resembles a stylized DNA double helix or infinity symbol
- Faceted, polygonal surface (not smooth curves)
- Single continuous band that folds back on itself, creating an open loop
- Tapered ends into sharp points
- Strong sense of three-dimensionality and volume through shading

**Wireframe Elements:**
- Delicate, light grey wireframe network surrounding the main shape
- Thin grey lines connecting small circular nodes
- Some lines extend beyond the red shape boundaries
- Creates depth and spatial framework
- Very subtle, not competing with main shape

**Layout Constraints (IMPORTANT):**
- The image will be used as a background for a two-column layout
- Left 40-50% of the image: Text content area (title, welcome message, button)
- Right center 40-50% of the image: Reservation form card (grey background card)
- Main visual elements (red 3D shape) should be positioned to avoid these content areas
- Recommended placement: Center-top, center-bottom, or left/right edges
- Keep central area (where text/card will be) relatively clear or very subtle

**Technical Requirements:**
- High resolution: 1920×1080px or larger
- Professional quality, suitable for web use
- Subtle enough to not interfere with text readability
- Can be slightly desaturated or have reduced opacity if needed
- No text or typography in the image
- No recognizable objects or people

**Mood & Feel:**
- Professional and trustworthy
- Modern and innovative
- Clean and uncluttered
- Sophisticated and premium
- Scientific and precise
```

## 프롬프트 (한글 - AI 이미지 생성 도구용)

```
제약회사 방문예약 웹사이트용 모던하고 깔끔한 추상 배경 이미지를 생성해주세요.

**스타일 & 미학:**
- 모던하고 미니멀하며 전문적인 디자인
- 추상적인 3D 기하학적 형태
- 로우폴리 또는 면으로 구성된 다각형 표면
- 미묘한 와이어프레임 네트워크 요소
- 깔끔한 흰색 또는 매우 밝은 배경
- 디지털적, 건축적, 과학적인 미학

**컬러 팔레트:**
- 주 색상: 밝은 빨간색 (#FF1A00, rgb(255, 26, 0)) - "KD Red"
- 다양한 빨간색 음영 사용: 밝은 오렌지-빨강 하이라이트, 깊은 어두운 빨강 그림자
- 빨간색 표면에 걸친 미묘한 그라데이션
- 연한 회색 와이어프레임 선과 노드 (매우 미묘함)
- 순수한 흰색 또는 약간 크림색 배경 (#FFFFFF 또는 #FAFAFA)

**주요 요소:**
- 중앙 3D 기하학적 형태: 꼬이고 루프를 만드는 연속적인 리본 형태
- 형태는 스타일화된 DNA 이중 나선 또는 무한대 기호를 닮음
- 면으로 구성된 다각형 표면 (부드러운 곡선 아님)
- 자신을 접어 열린 루프를 만드는 단일 연속 밴드
- 끝부분이 뾰족한 점으로 좁아짐
- 음영을 통한 강한 3차원성과 볼륨감

**와이어프레임 요소:**
- 주요 형태를 둘러싼 섬세하고 연한 회색 와이어프레임 네트워크
- 작은 원형 노드를 연결하는 얇은 회색 선
- 일부 선은 빨간색 형태의 경계를 넘어 확장
- 깊이와 공간적 프레임워크 생성
- 매우 미묘하며 주요 형태와 경쟁하지 않음

**레이아웃 제약사항 (중요):**
- 이미지는 2열 레이아웃의 배경으로 사용됨
- 이미지의 왼쪽 40-50%: 텍스트 콘텐츠 영역 (제목, 환영 메시지, 버튼)
- 이미지의 오른쪽 중앙 40-50%: 예약 폼 카드 (회색 배경 카드)
- 주요 시각적 요소(빨간색 3D 형태)는 이러한 콘텐츠 영역을 피하도록 배치
- 권장 배치: 중앙 상단, 중앙 하단, 또는 좌우 가장자리
- 중앙 영역(텍스트/카드가 위치할 곳)은 상대적으로 비워두거나 매우 미묘하게 유지

**기술적 요구사항:**
- 고해상도: 1920×1080px 이상
- 웹 사용에 적합한 전문적인 품질
- 텍스트 가독성을 방해하지 않을 정도로 미묘함
- 필요시 약간 탈채도 처리되거나 불투명도가 낮아질 수 있음
- 이미지에 텍스트나 타이포그래피 없음
- 인식 가능한 객체나 사람 없음

**무드 & 느낌:**
- 전문적이고 신뢰할 수 있음
- 모던하고 혁신적
- 깔끔하고 정돈됨
- 세련되고 프리미엄
- 과학적이고 정밀함
```

## 이미지 생성 후 확인사항

1. **텍스트 가독성 테스트**
   - 왼쪽 영역에 검은색 텍스트가 잘 보이는지 확인
   - 필요시 오버레이 추가

2. **카드 가독성 테스트**
   - 오른쪽 회색 카드가 이미지 위에서 잘 보이는지 확인
   - 카드 배경의 반투명도 조정 가능

3. **반응형 확인**
   - 모바일(세로 배치)에서도 잘 작동하는지 확인
   - 이미지가 모든 화면 크기에서 적절히 표시되는지 확인

4. **파일 최적화**
   - WebP 형식으로 변환 (더 작은 파일 크기)
   - 필요시 이미지 압축

## 파일 저장 위치

생성된 이미지는 다음 위치에 저장:
- `public/images/hero-background.webp` (또는 `.jpg`, `.png`)
- 또는 `src/assets/hero-background.webp`
